# Weihao Lee — Software Engineering Portfolio

Personal portfolio for internship recruiters. React (Vite), plain CSS, static deploy.

## Stack

- **React 19** + **Vite 6**
- Hand-written CSS (OKLCH tokens in `src/styles/`)
- Project assets in `public/assets/`

## Local development

```bash
npm install
npm run dev
```

Open the URL Vite prints (default `http://localhost:5173`).

## Production build

```bash
npm run build
npm run preview
```

Output goes to `dist/`. Deploy that folder to any static host (GitHub Pages, Netlify, Cloudflare Pages, etc.).
